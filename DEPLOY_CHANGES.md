# Как добавить новые файлы на хостинг Render

## Простой способ (через терминал)

### Шаг 1: Добавить изменения в Git

Откройте терминал на Mac и выполните команды:

```bash
# Перейти в папку проекта
cd /Users/romanbilskiy/Documents/Magnet_app

# Посмотреть, какие файлы изменены
git status

# Добавить все изменения (новые и измененные файлы)
git add .

# Создать коммит с описанием
git commit -m "Добавлена админ-панель"

# Отправить на GitHub
git push
```

### Шаг 2: Render автоматически задеплоит

После `git push`:
1. Render автоматически обнаружит изменения
2. Начнется автоматический деплой (2-3 минуты)
3. Ваши изменения появятся на сайте

---

## Проверка статуса деплоя

1. Зайдите в Render Dashboard: https://dashboard.render.com
2. Выберите ваш Frontend сервис
3. Посмотрите вкладку "Events" - там будет видно процесс деплоя
4. Дождитесь статуса "Live" (зеленый)

---

## Если нужно задеплоить вручную

Если автоматический деплой не запустился:

1. В Render Dashboard выберите ваш сервис
2. Нажмите "Manual Deploy" (вверху справа)
3. Выберите "Deploy latest commit"
4. Дождитесь завершения

---

## Что происходит при деплое

**Frontend:**
- Render скачивает код с GitHub
- Устанавливает зависимости (`npm install`)
- Собирает проект (`npm run build`)
- Публикует собранные файлы

**Backend:**
- Render скачивает код с GitHub
- Устанавливает зависимости (`npm install`)
- Запускает сервер (`npm start`)

---

## Важные моменты

⚠️ **После изменения кода ВСЕГДА делайте:**
1. `git add .` - добавить изменения
2. `git commit -m "описание"` - создать коммит
3. `git push` - отправить на GitHub

✅ **Render автоматически задеплоит изменения!**
