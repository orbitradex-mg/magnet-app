# Исправление ошибки 404

## Проблема
Frontend пытается обратиться к `http://localhost:5000/api`, которого нет на сервере Render.

## Решение: Установить переменную окружения

### Шаг 1: Найдите имя вашего Backend сервиса

1. Зайдите в Render Dashboard: https://dashboard.render.com
2. Посмотрите список ваших сервисов
3. Найдите сервис типа **"Web Service"** (не "Static Site")
4. Запомните его имя (например: `magnet-backend-yosa`)

### Шаг 2: Установите переменную окружения в Frontend

1. В Render Dashboard выберите ваш **Frontend** сервис (`magnet-frontend`)
2. Перейдите в раздел **"Environment"** (слева в меню)
3. Нажмите **"Add Environment Variable"**
4. Заполните:
   - **Key:** `REACT_APP_API_URL`
   - **Value:** `https://ваш-backend-name.onrender.com/api`
   
   ⚠️ **Замените `ваш-backend-name` на реальное имя вашего backend!**
   
   Например, если backend называется `magnet-backend-yosa`, то:
   - **Value:** `https://magnet-backend-yosa.onrender.com/api`

5. Нажмите **"Save Changes"**

### Шаг 3: Пересоберите Frontend

**ВАЖНО:** После изменения переменных окружения нужно пересобрать приложение!

1. В Render Dashboard выберите ваш Frontend сервис
2. Нажмите **"Manual Deploy"** (вверху справа)
3. Выберите **"Deploy latest commit"**
4. Дождитесь завершения деплоя (2-3 минуты)
5. Обновите страницу в браузере (Cmd+R)

### Шаг 4: Проверьте

1. Откройте сайт
2. Нажмите F12 (Cmd+Option+I)
3. В консоли должно быть: `API URL: https://ваш-backend-name.onrender.com/api`
4. Попробуйте войти снова

---

## Как проверить, что Backend работает?

Откройте в браузере URL вашего backend:
- Например: `https://magnet-backend-yosa.onrender.com`

Должна быть ошибка 404 - это нормально, значит сервер работает!

Попробуйте также:
- `https://ваш-backend-name.onrender.com/api/auth/me`
- Должна быть ошибка "Access token required" - это нормально, значит API работает!
