# Быстрый старт

## Шаг 1: Установка зависимостей

Выполните команду в корневой директории проекта:

```bash
npm run install-all
```

Эта команда установит зависимости для:
- Корневого проекта
- Backend сервера
- Frontend клиента

## Шаг 2: Запуск приложения

Запустите сервер и клиент одновременно:

```bash
npm run dev
```

Или запустите отдельно в разных терминалах:

**Терминал 1 (Backend):**
```bash
npm run server
```

**Терминал 2 (Frontend):**
```bash
npm run client
```

## Шаг 3: Открыть в браузере

Откройте браузер и перейдите по адресу:
```
http://localhost:3000
```

## Что дальше?

1. Вы увидите список заказов
2. Нажмите на заказ #1825, чтобы увидеть процессы
3. Нажмите "Приступить к выполнению" на любом процессе
4. После начала работы нажмите "Завершить" для завершения процесса
5. Когда все процессы завершены, появится кнопка "Готово - Завершить заказ"

## Примечания

- Backend работает на порту 5000
- Frontend работает на порту 3000
- База данных SQLite создается автоматически при первом запуске
- Тестовый пользователь и заказ создаются автоматически
